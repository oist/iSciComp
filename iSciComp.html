Aim of this Book#

This book aims to provide students from non-computational backgrounds with the basic knowledge and practical skills of computing that are required in almost all fields of science today. It also tries help students’ intuitive understanding of basic mathamatical concepts like eigenvalues and stability through computation and visualization. Python is used as the standard programming language, but the concepts covered can be helpful also in using other computing tools for data analysis and simulation.
